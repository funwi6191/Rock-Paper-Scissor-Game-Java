
# ✊✋✌️ Rock-Paper-Scissors Game (Java)

A simple command-line Rock-Paper-Scissors game built with Java. The player competes against the computer, which randomly selects its move.

## 🚀 Features
- Play Rock, Paper, or Scissors against the computer
- Randomized computer choice
- Displays winner (player, computer, or draw)
- Tracks number of rounds played
- Beginner-friendly Java project
